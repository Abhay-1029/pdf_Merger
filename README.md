# PDF Merger

A simple Python utility that merges multiple PDF files into a single PDF document using the PyPDF2 library.

## Features

* Merge multiple PDF files
* Easy command-line interface
* Fast and lightweight

## Requirements

```bash
pip install PyPDF2
```

## Usage

```bash
python main.py
```

Enter the PDF filenames when prompted, and the program will generate a merged PDF file.
